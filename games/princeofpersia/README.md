#          PRINCE OF PERSIA (1990)                     
*Incarnez le Prince et sauvez la princesse du palais   
avant que le vizir ne remporte la victoire !*      



### OBJECTIF DU JEU
----------------
Traversez le palais, évitez les pièges et les gardes,
et sauvez la princesse. Vous disposez de 60 minutes.

### COMMANDES
```bash
Déplacements :
  <- : avancer à gauche
  -> : avancer à droite
  ↑  : sauter
  ↓  : accroupir / descendre d'une plateforme

Combat :
  SHIFT gauche (ou CTRL) : dégainer l'épée / attaquer

Autres :
  ESC : pause / quitter
``` 
#### ASTUCES

- Observez et mémorisez les pièges.
- Utilisez les murs pour sauter plus haut.
- Attaquez les gardes au bon moment.
- Les sauvegardes sont limitées : préparez-vous à recommencer.

##### BONNES CHANCES, PRINCE !

Que vos réflexes et votre courage vous guident à travers
ce palais périlleux et jusqu'à la princesse !
